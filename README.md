# Languages API

This is a project to learn about Restful API's, Spring Boot, Kotlin and, maybe, Spring Security

### Clone
```bash
git clone https://github.com/julioomoura/languages-api.git
```
### Running 

```bash
./gradlew bootRun
```

### Build

```bash
./gradlew clean build
```

### Info
- Spring Boot
- Spring Data
- Gradle
- Kotlin
- MySQL
